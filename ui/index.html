📦 1. Tạo sản phẩm mới (single image)






🖼️ 2. Upload gallery (tối đa 5 ảnh)


📌 Lưu ý: Tối đa 5 ảnh cùng lúc (middleware: uploadCloud.array("images", 5))

Thông tin:
- Endpoint tạo sản phẩm: POST /api/products (middleware: uploadCloud.single("image") → validate → handler)
- Endpoint upload gallery: POST /products/gallery (middleware: uploadCloud.array("images", 5))
- Nếu không chọn file nào ở gallery → server trả về 400: "Vui lòng chọn ít nhất 1 ảnh"
